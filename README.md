# Godot 3D Pixel Art

Making 3D scenes in Godot look like pixel art.

Featuring:
- Pixel-perfect camera with smooth planar subpixel movement
- Stepped 3D animation playback to emulate spritesheet animation
- Stylized volumetric godrays

Inspired by [@t3ssel8r](https://x.com/t3ssel8r) ❤️
